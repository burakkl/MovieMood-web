# 🎬 MovieMood Web

MovieMood is a Netflix-inspired movie discovery and social platform built with React and Node.js. It is a complete rewrite of a Java Swing desktop application into a modern web application.

## ✨ Features

- **User System:** Registration, login, and secure session management.
- **Movie Discovery:** Browse popular movies, search by title, and view detailed information (cast, trailers, etc.).
- **Interaction:** Rate movies from 1 to 10 stars and add comments.
- **Favorites:** Save your favorite movies to your personal list, and generate new lists based on your suggestions.
- **Modern Design:** A responsive, dark-themed, and user-friendly interface.

## 🛠️ Tech Stack

- **Frontend:** React, React Router, Vite, CSS Modules
- **Backend:** Node.js, Express
- **Database:** SQLite

## 🚀 Quick Start

Ensure you have **Node.js** installed on your system before running the project.

**1. Install dependencies:**
```bash
npm install
cd server && npm install
cd ../client && npm install
```

**2. Start the application:**
The following command runs both the backend server and the frontend client concurrently:
```bash
npm run dev
```

**Application URLs:**
- Frontend: `http://localhost:5173`
- Backend API: `http://localhost:3000`
